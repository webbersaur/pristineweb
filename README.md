# Pristine Exteriors CT - Website Content

Content files for [Pristine Exteriors CT](https://pristineexteriorsct.com) WordPress website.

## Vinyl Siding Location Pages

Unique, SEO-optimized content for location-specific vinyl siding service pages:

| File | Town | Unique Angle |
|------|------|--------------|
| `siding-branford.md` | Branford | Coastal/shoreline focus, salt air protection, Stony Creek, historic village |
| `siding-hamden.md` | Hamden | Local HQ, diverse neighborhoods, Quinnipiac area, suburban family homes |
| `siding-north-haven.md` | North Haven | Classic suburb, property investment, convenient location |
| `siding-guilford.md` | Guilford | Historic architecture (1639), town green, coastal exposure, premium properties |

## Word Count
Each page contains approximately **650-700 words** of unique content to satisfy Rank Math SEO requirements (600+ words recommended).

## Content Structure
Each file includes:
- Hero section H1 heading
- Secondary H1 heading
- Intro section (3 paragraphs)
- "Why Invest" / importance section (3 paragraphs)
- Services list (5 bullet points)
- "Why Choose Us" section (3 paragraphs)

## Usage
Copy the content from each markdown file into the corresponding WordPress/Divi page using the Visual Builder or Classic Editor.
